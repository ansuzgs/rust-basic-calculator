# Rust Basic Calculator

Calculadora básica implementada en Rust que soporta la operación de suma (`+`) con entradas que contienen espacios.


## Compilación
```bash
cargo build --release
```


## Ejecucion
```bash
cargo run --release
```

## Pruebas
```bash
cargo test
```

## Ejemplos de uso
```bash
>> 3 + 5
= 8
```
